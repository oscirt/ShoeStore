# ShoeStore
Shoe store application with authorization and addition of new shoes to the list.

## Features
 - Authorization
 - Shoe store description
 - List of shoes
 - Description of shoes

## Stack
 - Kotlin
 - MVVM
 - Fragments
 - Jetpack Navigation
